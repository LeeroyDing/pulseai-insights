# Documentation Sitemap

This directory contains the blueprint and verification for the PulseAI Insights project.

## Files

1. **[FEASIBILITY_STUDY.md](./FEASIBILITY_STUDY.md)**
   - *Abstract:* A critical analysis of the technical and economic viability of the automated loop. Focuses on API costs vs. projected revenue and data source reliability.

2. **[TECHNICAL_SPEC.md](./TECHNICAL_SPEC.md)**
   - *Abstract:* Detailed system architecture. Defines the data flow between the Scout (scraper), Editor (LLM), and Publisher (SSG/Social APIs).

3. **[MONETIZATION_PLAN.md](./MONETIZATION_PLAN.md)**
   - *Abstract:* Deep dive into automated revenue streams. Details the logic for affiliate link insertion and programmatic ad integration.

4. **[OPERATIONAL_GUIDE.md](./OPERATIONAL_GUIDE.md)**
   - *Abstract:* The 'Manual' for the AI manager. Defines how the P&L is tracked, how errors are handled, and how the system self-optimizes.
